# 🚀 Deploy remove.khanjansevakendra.shop

One-command deployment to GitHub and Railway.

## ⚡ Quick Deploy

### Step 1: Run the Deployment Script

```bash
cd /Users/arbaz/Downloads/rembg-main
./DEPLOY_EVERYTHING.sh
```

This script will:
- ✅ Push code to GitHub
- ✅ Guide Railway deployment
- ✅ Configure domain
- ✅ Setup DNS
- ✅ Update OAuth settings

### Step 2: Follow the Prompts

The script will ask you to:
1. Enter GitHub repository URL
2. Configure Railway (opens browser)
3. Add environment variables
4. Set up domain
5. Configure GoDaddy DNS
6. Update Google OAuth
7. Setup Instamojo webhook

### Step 3: Done! 🎉

Your site will be live at:
- **https://remove.khanjansevakendra.shop**

---

## 📋 Pre-Deployment Checklist

Before running the script, make sure you have:

- [ ] GitHub account: https://github.com/join
- [ ] Railway account: https://railway.app/ (Login with GitHub)
- [ ] GoDaddy access: https://dcc.godaddy.com/
- [ ] Google Cloud Console: https://console.cloud.google.com/
- [ ] Instamojo Dashboard: https://www.instamojo.com/
